Working with CSS

Tips for Organizing CSS

  • Use comments liberally
  • User consistent naming conventions
    • As in how you name your selectors - are you using dashes, underscores, camel case?)
  • Use consistent formatting
    • Be consistent in your spacing, tabs, whether the rule is all on one line or on multiple lines, etc.
  • Group like styles whenever possible. For example, group all the h1, h2, etc. in the same area.
  • Split up files

Browser Inspection Tools

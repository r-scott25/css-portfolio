CSS Fonts

Basic Font Properties

  • Font family (font-family)
  • Font weight (font-weight)
  • Font style (font-style)
  • Font size (font-size)

Font Family

  • Specifies the font for an element
  • Is a prioritized list of fonts
  • Includes a fallback font
    • p { font-family: "Times New Roman", Times, serif; }

Font Weight

  • Specifies the weight of a font
    • p { font-weight: bold;}
    • normal, bold, bolder, lighter, 100, 200, . . . 900
  • Not all weights are available for all font families

Font Style

  • Specifies the style of a font
  • normal, italic, oblique
  • Not all styles are available for all font families

Font Size

  • Specifies the size of a font
  • absolute-size, relative-size, length-percentage

Font Options

  • System Fonts
    • aka fallback fonts
    • Pre-installed on almost every machine
  • Web Fonts
    • Google Fonts (free)
    • Adobe Fonts (require a subscription
    • External style sheets that you can include in your project using these methods:
      • @font-face = CSS font face rule:
        • Allows you to download a font from your web server to render a webpage if the user doesn't have that font installed
      • @import = import rule
        • Can be used from an existing CSS file to import an external font resource
        • Not recommended b/c not as efficient for performance
      • Link to an exteral style sheet
      • Load via JavaScript
  • Note: Be mindful that for every font style and weight that you load on a site, there's a corresponding performance implication

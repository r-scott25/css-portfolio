CSS Margins and Padding

Padding

  • The clear/transparent space that sits between the edge of an element and its border
  • Individual Properties
    • padding-top: 10px
    • padding-right: 20px
    • padding-bottom: 10px
    • padding-left: 20px
  • Shorthand Notation
    • padding: 10px 20px 15px 10px;
      • These values apply in the order of top, right, bottom, left
    • padding: 10px 20px 15px;
      • These values apply in the order of top, right/left, bottom
    • padding: 10px 20px;
      • These values apply in the order of top/bottom, right/left
    • padding: 10px;
      • All sides share the same value top/bottom/left/right

Margins

  • Applied outside of an element's border, and represent the space between elements
  • While not factored into the total width of an element, it does affect the amount of space that an element takes up on a page
  • Many elements have a default margin defined by user agents (If you use a CSS reset, to set your margins to 0, then you don't have to worry about it
  • Individual Properties
    • margin-top: 10px
    • margin-right: 20px
    • margin-bottom: 10px
    • margin-left: 20px
  • Shorthand Notation
    • margin: 10px 20px 15px 10px;
      • These values apply in the order of top, right, bottom, left
    • margin: 10px 20px 15px;
      • These values apply in the order of top, right/left, bottom
    • margin: 10px 20px;
      • These values apply in the order of top/bottom, right/left
    • margin: 10px;
      • All sides share the same value top/bottom/left/right
  • Note: Unlike horizontal margins, vertical margins are somtimes combined, or collapsed, into a single margin whose size is the largest of the individual margins.That means that if you have two elements stacked on top of one another, only one of the elements' top and bottom margins are applied.

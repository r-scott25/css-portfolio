Measurement

Four Most Common Units of Measurement in CSS

  • px = pixels
  • em
  • rem
  • % = percentages

px

  • Absolute (fixed) value length -- Just like an inch takes up a physical amount of space, so does a pixel
  • Used to define height and width values

em

  • Relative value (relative to the font size of the parent element)
  • Value of text at its default size
    • Example: If you set your parent container to a 20px font size, that would be equal to 1 em. Two ems would be 40px, 3 ems would be 60px and so on. From there, you could set your paragraphs to 1 em, maybe your headings to 3 em, and you can even use the em value for other properties like margin or padding.
    • You can change the starting point of the parent element, and the overall size of em units will get bigger or smaller as you've designated, but the relationship between those elements will stay exactly the same.

rem (root ems)

  • Relative value (relative to the font size of the root element, which is <html>)
  • Only value of root is considered, not parent element
  • Offers more predictable control than em
  • If the HTML element doesn't have a specified font size, the browser default of 16 pixels is used.
    • 1 rem = 16px, 2 rem = 32 px, etc.

%

  • Relative to the parent element
  • Used in responsive layouts
  • Not typically used for font sizing
  • Typically used on block-level elements

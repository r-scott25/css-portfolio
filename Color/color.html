Color Keywords

  • Keywords map to a predetermined set of values that browsers use to display the color

RGB

  • Generates all the colors from the combination red, green, and blue
  • rgb(num1, num2, num3)
  • Color intensities are measured on a 0 to 255 scale, where 0 measures the absence of that color, and 255 is the highest
  • rgb(0, 0, 0) = black (the absence of all colors)
  • rgb(255, 255, 255) = white (all colors at their highest intensity)

Hexadecimal (Hex) Notation

  • Another way to represent RGB
  • Noted by a # sign followed by three hex pairs representing the red, green, and blue values of a color
  • Example: Red = #ff0000 (ff for red, 00 for no green, and 00 for no blue)

HSL

  • Hue, Saturation, Lightness
  • Example: hsl(0, 100%, 50%) (0 for Hue, 100% for Saturation, 50% for lightness)
  • A color's hue is represented as an angle on the color wheel, which ranges from zero to 360. Red is 0, Green is 120, Blue is 240
  • The saturation represents the amount of color, with 100% being fully saturated and 0% being gray scale.
  • Lightness tells how bright or dark the color is.

RGBa and HSLa

  • red, green, blue, alpha
  • hue, saturation, light, alpha
  • Alpha specifies the opacity of a color on a scale of 0 to 1, where 0 is fully transparent and one is full opaque
